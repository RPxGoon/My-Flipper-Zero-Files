By: RPxGoon

This script opens PowerShell as an administrator and then removes several Windows bloatware applications using the Remove-AppxPackage cmdlet. 
The applications that will be removed by this script are:

3D Builder
Get Help
Get Started
People
Skype
OneNote
Maps
Microsoft Solitaire Collection
Mixed Reality Portal
Sway
Xbox
Feedback Hub
Weather
Alarms & Clock
Camera
Communications Apps
Please note that running this script will remove these applications permanently from your system,
so make sure you backup any data you may have stored in them before running it. Also, depending on your system configuration, 
you may need to modify the script or run it manually to ensure it works as intended.
